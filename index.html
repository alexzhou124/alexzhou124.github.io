What have I done?

  • Employee Feedback Tool

    Worked as part of an agile group to create modular and scalable full stack employee feedback tool for Ultimate Software using a React, Flask, MongoDB stack. Allowed for reading, requesting and writing reviews. Pulled employee data and automatically generated login credentials for hundreds of employees. Created an intuitive UI with accessibility tools incorporated.

  • Stock Bot

    A bot written in Python to continuously check a website for an item to be in stock. Will then email and and text me and will check out using Selenium. Just pray there's no captcha.

  • Connect 4 with AI

    A variation of Connect 4 written in Python where the board can be variable sized. Created an AI opponent using minimax and alpha-beta pruning to be efficient while maintaining accuracy. Wrote a heuristic function that determines the rankings for each potential state.
